About Boilerplate

Login Component

When user run this boilerplate, they will be routed to to this login page where they have to enter "Username" and "Password". If they haven't register then, a swal pop-up will be fired and after, clicking on "Ok" button, they will be routed to register page.

Dashboard

After login, user will be routed to dashboard page and use different commands to create new components, services, angular material, pwa support, run and watch test cases and build for production.

Details

Showing data of airlines.
It has search, filter and pagination functionality.


User can also clear filers and choose items per page.

Charts

It has different types of Charts.

1.Bar Chart
2.Line Chart
3.Pie Chart
4.Donut Chart

Pipes

Different types of pipes we are showing to transform data. For example:
date, uppercase, lowercase, title, currency, percent, slice and decimal pipe

Loaders

It has two types of loaders.

1. Full page loader
2. Component loader

Forms

It has two types of forms.

1.Reactive form
2.Template Driven forms

Data Sharing

Sharing data between components using
1.Input
2.Output
3. View child
4.Behaviour Subject.

Error Handling

Showing different types of error templates
1.404
2.502
3.400
4. Others Error
